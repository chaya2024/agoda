# 🚀 הוראות התקנה מהירות

## למה אתה רואה מסך לבן?

הסיבה היא שהאתר מנסה להתחבר ל-Supabase (מסד הנתונים) אבל אין עדיין מפתחות אמיתיים.

## 📝 פתרון בשני שלבים

### שלב 1: צור פרויקט Supabase (5 דקות)

1. **היכנס ל-Supabase**
   - לך ל-[supabase.com](https://supabase.com)
   - לחץ על "Start your project"
   - התחבר עם GitHub או Google

2. **צור פרויקט חדש**
   - לחץ על "New Project"
   - תן שם לפרויקט (לדוגמה: "student-union")
   - בחר סיסמה חזקה
   - בחר אזור (Europe West המומלץ)
   - לחץ "Create new project"
   - חכה 2 דקות עד שהפרויקט מוכן

3. **הרץ את קוד מסד הנתונים**
   - בצד שמאל, לחץ על "SQL Editor"
   - לחץ על "+ New query"
   - פתח את הקובץ `supabase/migrations/20250110_create_student_union_schema.sql`
   - העתק את **כל** התוכן
   - הדבק בחלון ה-SQL
   - לחץ "Run" (או Ctrl+Enter)
   - אמור לראות "Success. No rows returned"

4. **קבל את המפתחות**
   - בצד שמאל, לחץ על ⚙️ "Settings"
   - לחץ על "API"
   - תראה שני דברים חשובים:
     - **Project URL** - משהו כמו `https://abcdefgh.supabase.co`
     - **anon public** - מפתח ארוך מאוד

### שלב 2: הגדר את המפתחות בפרויקט

1. **ערוך את קובץ ה-.env**
   ```bash
   cd frontend
   ```

2. **פתח את `.env` בעורך טקסט** והחלף את הערכים:
   ```
   VITE_SUPABASE_URL=https://YOUR-PROJECT-ID.supabase.co
   VITE_SUPABASE_ANON_KEY=YOUR-LONG-ANON-KEY-HERE
   ```

3. **שמור את הקובץ**

### שלב 3: הרץ את האתר

```bash
npm run dev
```

פתח דפדפן בכתובת: `http://localhost:5173`

**עכשיו אתה אמור לראות את האתר! 🎉**

## 🐛 עדיין רואה מסך לבן?

### בדיקה 1: משתני סביבה
```bash
cat frontend/.env
```
וודא שהמפתחות מתחילים ב-`VITE_` ושהם לא "placeholder" או "demo"

### בדיקה 2: הפעל מחדש את שרת הפיתוח
```bash
# עצור את השרת (Ctrl+C)
npm run dev
```

### בדיקה 3: פתח את Console בדפדפן
- לחץ F12
- עבור ל-"Console"
- חפש שגיאות באדום
- אם יש שגיאת חיבור ל-Supabase, בדוק את המפתחות שוב

### בדיקה 4: בדוק שהטבלאות נוצרו
- חזור ל-Supabase Dashboard
- לחץ על "Table Editor" בצד שמאל
- אמור לראות 7 טבלאות:
  - departments
  - news
  - gallery_images
  - store_products
  - rights
  - team_members
  - users

## 💡 טיפים

### רוצה לערוך את התוכן?
כל התוכן נמצא ב-Supabase:
1. לך ל-Table Editor
2. בחר טבלה (לדוגמה `news`)
3. לחץ על שורה לעריכה
4. שמור
5. רענן את האתר - התוכן עודכן!

### רוצה להוסיף תמונות?
1. ב-Supabase, לחץ על "Storage"
2. צור bucket חדש בשם "images"
3. העלה תמונות
4. קבל את ה-URL
5. הוסף אותו לשדה `image_url` בטבלה
